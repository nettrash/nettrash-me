# nettrash.me

A developer utility toolkit built with **Rust** + **WebAssembly** (Yew framework), served by **nginx**.

## Features

- **Home** — Client info: IP address, local date/time, UTC, geolocation
- **Math** — Hash (MD5, SHA1, SHA256, SHA384, SHA512), Luhn validation, GUID generation
- **Text** — Base64, URL, Hex encode/decode, RegEx matching
- **Time** — Unix timestamp conversion

Everything runs **client-side in WebAssembly**. nginx serves the static files and provides the client IP via a tiny `/api/ip` endpoint.

## Prerequisites

- [Rust](https://rustup.rs/) (stable)
- `wasm32-unknown-unknown` target: `rustup target add wasm32-unknown-unknown`
- [Trunk](https://trunkrs.dev/): `cargo install trunk`
- nginx

## Build

```bash
cd frontend && trunk build --release
```

## Deploy

```bash
# Copy built files to nginx web root
cp -r dist/* /var/www/nettrash.me/

# Copy and enable nginx config
cp nginx.conf /etc/nginx/sites-available/nettrash.me
ln -s /etc/nginx/sites-available/nettrash.me /etc/nginx/sites-enabled/
nginx -t && nginx -s reload
```

## Development

```bash
cd frontend && trunk serve
```

## Project Structure

```
Cargo.toml              # Workspace root
nginx.conf              # nginx site config
frontend/               # Yew WASM frontend
  Cargo.toml
  Trunk.toml
  index.html            #   HTML shell
  styles.css            #   Global styles
  src/
    main.rs             #   App entry, router
    components/
      nav_menu.rs       #   Navigation bar
      home.rs           #   Home page (client info)
      math.rs           #   Math tools (hash, luhn, guid)
      text.rs           #   Text tools (base64, url, hex, regex)
      time.rs           #   Time tools (unixtime)
dist/                   # Built WASM output (generated by trunk build)
```
